# 管理员登录说明

## 管理员账户信息

- **用户名**：一只炸虾
- **密码**：zhaxia
- **角色**：管理员

## 登录步骤

1. 打开登录页面
2. 在"登录类型"下拉菜单中选择 **"管理员登录"**
3. 在"账号"输入框中输入：**一只炸虾**
4. 在"密码"输入框中输入：**zhaxia**
5. 点击"立即登录"按钮

## 注意事项

- 必须选择"管理员登录"类型，否则会提示登录类型不匹配
- 管理员账户使用用户名登录，不是手机号或邮箱
- 如果忘记密码，可以运行 `src/backend/create_admin.py` 脚本重置密码

## 创建/重置管理员账户

如果需要重新创建或重置管理员账户，运行以下命令：

```bash
cd src/backend
python create_admin.py
```

这个脚本会：
- 如果管理员账户已存在，会更新密码为 `zhaxia`
- 如果管理员账户不存在，会创建新账户
- 确保账户角色为管理员（admin）

## 管理员功能

登录后，管理员可以：
- 查看所有用户列表
- 启用/禁用用户账户
- 创建咨询师账户
- 删除咨询师
- 查看系统日志
- 管理平台所有功能
